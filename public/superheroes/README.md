# Superhero Images

To add actual superhero images to the app:

1. Download superhero images (PNG format recommended)
   - superman.png
   - batman.png
   - spiderman.png
   - hulk.png
   - captain-america.png
   - ironman.png
   - flash.png
   - wonder-woman.png

2. Place the image files in this folder: `/public/superheroes/`

3. The app will automatically use these images. If images are not found, it will fall back to emoji icons.

**Note:** Make sure you have the rights to use these images. For educational purposes, you can find free superhero images on:
- Pixabay (pixabay.com)
- Pexels (pexels.com)
- Unsplash (unsplash.com)

Search for terms like "superman", "spiderman", "batman", etc.
